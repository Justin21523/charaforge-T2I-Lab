    Technical stack
    
          
    What this project demonstrates
    
        
    
        
    
          
    FrontendReact, Vite, react-router, lucide icons, localStorage state, dashboard UI
    
          
    BackendFastAPI, Pydantic settings, structured error format, CORS, GZip, request IDs
    
          
    AI pipelinePyTorch, Diffusers, PEFT, LoRA manager, ControlNet hooks, safety/watermark modules
    
          
    WorkersRedis-backed queues, Celery training tasks, cancellable jobs, cleanup endpoints
    
          
    DeploymentDocker Compose for local full stack; GitHub Pages for static portfolio review
